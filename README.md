# weekly-us

> 매주 개인 활동을 기록하고 공유하는 저장소입니다.

## 이슈 작성 가이드

매주 목요일 세션에서 발표할 내용을 **이슈**로 작성합니다.<br>
게시한 이슈를 화면에 띄워 **모든 팀원이** 발표하며, 한주동안 배운점을 5분 이내로 짧게 공유해주시면 됩니다.

### 규칙

- **주 1회**, 목요일 세션 전까지 이슈를 올려주세요.
- 이슈 제목은 `[n주차] 본인이름` 형식으로 작성합니다. (예: `[1주차] 김우진`)
- 이슈를 생성하면 양식이 자동으로 채워집니다.
- 성실하게 작성해주세요. 분량보다 **구체성**이 중요합니다.
